# LLM Engineering Roadmap

A hands-on learning journey from **Natural Language Processing (NLP)** to **Large Language Model (LLM) Engineering**, covering modern Generative AI concepts such as Transformers, Embeddings, Semantic Search, Vector Databases, Retrieval-Augmented Generation (RAG), LangChain, and AI Agents.

This repository documents my daily practical work, code implementations, notes, and mini-projects while following a structured roadmap toward becoming an LLM Engineer.

---

## Learning Goals

- Build a strong foundation in NLP
- Understand how Transformer-based LLMs work
- Learn text embeddings and semantic search
- Build vector databases using FAISS
- Implement Retrieval-Augmented Generation (RAG)
- Work with LangChain
- Integrate Google Gemini API
- Develop AI applications using modern LLM frameworks
- Progress toward AI Agent development
